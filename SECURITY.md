# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| Latest release | ✅ |
| Older releases | ❌ |

We support only the latest release. Please update before reporting a vulnerability.

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues privately via [GitHub Security Advisories](https://github.com/etorhub/farmaciola/security/advisories/new).

Include as much detail as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce
- Affected versions
- Any suggested remediation (optional)

You can expect an initial response within 72 hours. We will keep you informed of our progress and coordinate a disclosure timeline with you before publishing any fix.
